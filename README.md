# Progetto Ingegneria del Software - Frontend

Questo repository contiene il frontend del progetto per il corso di Ingegneria del Software.

## Tecnologie utilizzate

- React
- JavaScript/TypeScript
- CSS/SCSS
- [Altre tecnologie rilevanti]

## Installazione

1. Clona il repository:
  ```bash
  git clone https://github.com/tuo-utente/progetto-ingegneria-frontend.git
  ```
2. Installa le dipendenze:
  ```bash
  npm install
  ```
3. Avvia l'applicazione:
  ```bash
  npm start
  ```

## Struttura del progetto

- `/src`: codice sorgente dell'applicazione
- `/public`: file pubblici e statici
